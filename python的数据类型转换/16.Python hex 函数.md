# Python hex() 函数
## 描述
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;hex() 函数用于将10进制整数转换成16进制整数。

## 语法
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;hex 语法：

```python
hex(x)
```

## 参数
- x -- 10进制整数

## 返回值
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;返回16进制整数。

## 实例
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;以下实例展示了 hex 的使用方法：

```python
>>>hex(255)
'0xff'
>>> hex(-42)
'-0x2a'
>>> hex(1L)
'0x1L'
```
