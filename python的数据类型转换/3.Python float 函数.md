# Python float() 函数
## 描述
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;float() 函数用于将整数和字符串转换成浮点数。

## 语法
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;float()方法语法：

```python
class float([x])
```

# 参数
- x -- 整数或字符串

## 返回值
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;返回浮点数。

## 实例
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;以下实例展示了 float() 的使用方法：

```python
>>>float(1)
1.0
>>> float(112)
112.0
>>> float(-123.6)
-123.6
>>> float('123')     # 字符串
123.0
```
